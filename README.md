## Run Locally

Clone the project

```bash
  git clone https://github.com/GJVizconde/tickets_challenge_auth.git
```

Go to the project directory

```bash
  cd tickets_challenge_auth
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run deploy
```
